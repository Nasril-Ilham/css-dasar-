posision sticky

elemen yang menggunakan posision sticky akan berperilaku seperti posision relative hingga mencapai batas tertentu, setelah itu akan berperilaku seperti posision fixed. elemen dengan posision sticky tetap berada dalam aliran normal dokumen, sehingga mempengaruhi posisi elemen lain di halaman. properti top, bottom, left, dan right dapat digunakan untuk menentukan batas di mana elemen menjadi "lengket".

elemen ini diatur menggunakan posision sticky


















posision statis

position static digunakan untuk mengatur element menjadi statis secara default. element akan mengikuti posisi normal secara default elemen ini dapat diatur menggunakan properti top, bottom, left, dan right.

elemen ini diatur menggunakan posision statis


posision relative

sebuah elemen html yang diatur menggunakan posision relative akan tetap berada pada posisi normalnya, tetapi dapat digeser menggunakan properti top, bottom, left, dan right dari elemen yang di posisikan "relative" akan membuat jauh dari posisi normal. konten lain tidak akan disesuaikan agar sesuai dengan celah yang ditinggalkan oleh elemen yang diposisikan "relative".

elemen ini diatur menggunakan posision relative







posision absolute

elemen yang menggunakan posision absolute akan dihapus dari aliran normal dokumen dan diposisikan relatif terhadap elemen induknya yang memiliki posision relative. jika tidak ada elemen induk dengan posision relative, elemen tersebut akan diposisikan relatif terhadap jendela tampilan (viewport). elemen dengan posision absolute dapat diatur menggunakan properti top, bottom, left, dan right untuk menentukan posisinya. dan elemen dengan posision absolute tidak mempengaruhi posisi elemen lain di halaman. untuk mencenterkan elemen dengan posision absolute, kita bisa menggunakan kombinasi properti top, left, dan transform.

elemen ini diatur menggunakan posision relative
elemen ini diatur menggunakan posision absolute







posision fixed

sebuah elemen html yang diatur menggunakan posision fixed akan tetap berada pada posisi yang ditentukan, bahkan saat halaman digulir. elemen dengan posision fixed dihapus dari aliran normal dokumen, sehingga tidak mempengaruhi posisi elemen lain di halaman, berlaku pengaturan properti top, bottom, left, dan right pada elemen dengan posision fixed.

elemen ini diatur menggunakan posision fixed
